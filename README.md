# 🏨 Hotel Hives Dashboard

Hotel Hives is a modern hotel management and booking dashboard built with React, Vite, and Tailwind CSS.  
It helps hotel staff manage rooms, guests, and reservations easily through a clean, responsive, and fast interface.

---

## 🌐 Live Demo
🔗 [View Demo](https://hotel-hives.vercel.app/)  

---

## 🚀 Tech Stack
- ⚛️ React + Vite
- 🎨 Tailwind CSS
- 🔁 React Router DOM
- 🔍 React Query
- 🧾 React Hook Form
- 💾 Supabase
- 📊 Recharts

---

## 🧩 Features
- 📅 Real-time room and reservation management  
- 👥 Guest profiles and booking history  
- 💳 Secure payment flow  
- 📊 Dashboard analytics with charts and insights  
- 🌙 Light & Dark mode support  
- ⚙️ Simple, modern UI for admins and staff
